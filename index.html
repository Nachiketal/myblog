What's new ?

CLASSROOMS EQUIPPED WITH SMART BOARDS

All classes, from Nursery to P5, have been converted into Smart Board Digi Classes where teachers supplement curriculum with the installed resource data bank that is subject and class specific. Smart Boards make the teaching-learning process more effective for visual and auditory learners. Using technology, teachers are able to explain concepts and create a stimulating learning environment for students.

SPECIALISED HELP AND COUNSELLING

A centre for special education needs, called the Learning Centre, provides specialized help and a differentiated learning environment to children with learning challenges and cognitive difficulties. To ensure a progressive growth index in students, regular interactive sessions are held between stake holders. The school counsellors also handle all behavioural issues and address students during weekly classroom sessions and special assemblies.
